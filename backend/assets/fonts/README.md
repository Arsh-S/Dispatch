# PDF Report Fonts

The PDF report generator uses **Inter** (sans-serif) and **JetBrains Mono** (monospace) for improved typography. Both are OFL-licensed.

## Required Files

Place these TTF files in this directory:

- `Inter-Regular.ttf`
- `Inter-Bold.ttf`
- `Inter-SemiBold.ttf`
- `JetBrainsMono-Regular.ttf`

## Download Instructions

### Inter

1. Visit https://rsms.me/inter/ or https://fonts.google.com/specimen/Inter
2. Download the font family
3. Extract and copy `Inter-Regular.ttf`, `Inter-Bold.ttf`, `Inter-SemiBold.ttf` to this folder

### JetBrains Mono

1. Visit https://www.jetbrains.com/lp/mono/ or https://fonts.google.com/specimen/JetBrains+Mono
2. Download the font family
3. Extract and copy `JetBrainsMono-Regular.ttf` to this folder

## Fallback

If font files are not present, the PDF generator falls back to built-in Helvetica and Courier.
